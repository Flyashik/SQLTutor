Урок 8. Оператор DELETE

Оператор DELETE используется для удаления строк таблицы (ВАЖНО! Не для удаления самой таблицы)

DELETE FROM "таблица"
   [WHERE "условие поиска"]

Если не указано предложение WHERE, то будут удалены все строки таблицы.

Пример. Удалить из списка стран строку, относящуюся к России.

DELETE FROM COUNTRY
   WHERE COUNTRY = ’RUSSIA’;

Пример. Удалить из таблицы SALARY_HISTORY все сведения об изменении оплаты сотруднику с номером 65.

DELETE FROM SALARY_HISTORY
   WHERE EMP_NO = 65;

Пример. Очистить таблицу SALES_ARH.

DELETE FROM SALES_ARH;
